GitHub で緑化活動する

GitHub

エンジニアなら皆様ご存知のアレ。

  • コードを公開すると、他人がそれを複製・修正したりする。
  • 他人とコードを通してつながる、いわゆるソーシャルネットワーク。

GitHub のマスコット、 Octocat

Contributions Calendar について話します

緑化活動 ― GitHub で何か貢献すること

活動した日は、"Contributions calendar" が緑色に染まる！

  • リポジトリにコミットしてプッシュする。
  • バグ報告、機能改善要望などの "Issue" をたてる。
  • バグ修正、機能追加などの"Pull request" をなげる。

緑化活動をはじめる上で決めたルール

  • ゴミみたいなコードでもとりあえず上げる。
  • 時々、一切プログラミングしない日をつくる。 → ビールでも飲みに行こう。
  • やる気の出ない → 設定ファイル一行程度の変更でも OK

もっと非エンジニアにも関心を持ってもらえるモノを作りたい。

まとめ ― 緑化活動、しませんか

  • 緑化活動 (GitHub 上で活動すること) しませんか。
    • 普段触れない技術に触れる。
    • アウトプットの習慣をつくる。
  • 続けるのは案外簡単。続ければ何かしらできてくる。
    • mitsuse/bombay-vim
    • mitsuse/perceptron-go
    • mitsuse/kompl
  • 今後はもっと人に使ってもらえるモノをつくりたい。
